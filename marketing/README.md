# ApexBuild — Launch & Marketing Kit

Distribution-ready assets for **ApexBuild**, the AI Shopify app & merchant-tool
generator. Everything here is on-brand with the product (palette and type pulled
straight from `src/index.css` / `tailwind.config.ts`).

## Contents
- `assets/` — vector identity: app icon, wordmark, Open Graph card, feature graphic.
- `landing.html` — a self-contained, production-quality launch page (open in a browser).
- `copy/` — positioning, Shopify App Store listing, Product Hunt kit, launch thread, announcement email.

## Brand foundations

**Voice:** confident, concrete, builder-to-builder. We sell outcomes ("ship an
installable app"), never hype. Short sentences. Show the work.

**Palette** (dark-first):
| Token | Hex | Use |
|-------|-----|-----|
| Canvas | `#0D0E12` | backgrounds |
| Surface | `#15171D` | cards / panels |
| Primary (cyan) | `#22D3EE` | primary actions, accents |
| Indigo | `#5B50E0` | gradient mid |
| Violet | `#A06FD6` | gradient end |
| Foreground | `#E6EAF0` | text |
| Muted | `#7E8794` | secondary text |

**Signature gradient:** `135deg, #22D3EE → #5B50E0 → #A06FD6` (cyan→indigo→violet).
Use on the mark, key headlines, and one hero element per screen — never on body text.

**Type:** Space Grotesk (display/headlines), Inter (body). Tight tracking on
large display; generous line-height on body.

**Texture:** soft radial "glow" blobs of the gradient at low opacity behind dark
glass cards. Rounded corners (12–24px). Subtle 1px `rgba(255,255,255,0.08)` borders.

## Positioning one-liner
> **Describe it. Ship a Shopify app.** ApexBuild turns a sentence into an
> installable, Built-for-Shopify app — App Bridge, Polaris, Admin GraphQL,
> webhooks, and billing wired in — grounded in your real store.
